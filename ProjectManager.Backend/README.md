# ProjectMananger.BackEnd

## Tecnologias Utilizadas

- dotnet core >= 3.0
- Postgresql >= 12

### Restaurar Dependencias
```
dotnet restore
```

### Compilar projeto
```
dotnet build
```

### Compilar docker
```
docker build .
```

### Rodar testes
```
dotnet test
```

### Rodar projeto
```
dotnet run --project TaskManager.api/TaskManager.api.csproj
```

### publicar projeto para produção
```
dotnet publish --release
```
